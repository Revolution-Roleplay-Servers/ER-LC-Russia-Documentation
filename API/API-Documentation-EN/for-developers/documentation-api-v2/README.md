# 🚀 Документация API V2

Этот API предоставляет программный доступ к данным и функциям, связанным с сервером ER:LC и статистикой. Он разработан для интеграции с вашими собственными приложениями и сервисами.

<h3 align="center">Структура Документации</h3>

1. [mrp-erlc-api-v2](mrp-erlc-api-v2/ "mention")
   * Подробное описание всех доступных маршрутов API.
   * Методы, параметры, примеры запросов и ответов на JavaScript, Python и cURL.
   * Информация по аутентификации для разных групп эндпоинтов.
2. [error-codes.md](../error-codes.md "mention")
   * Список всех возможных кодов ошибок.
   * Описание причин и способы их решения.
   * Инструкции по действиям в случае неизвестных ошибок.
3. [rate-limits.md](../rate-limits.md "mention")
   * Список всех возможных Рейт Лимитов.
   * Описание причин и способы их решения.
