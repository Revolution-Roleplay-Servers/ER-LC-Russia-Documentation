# 🚨 Лимиты Запросов

{% hint style="warning" %}
## E**R:LC Россия API не поддерживает хостинги ботов, такие как Bot Ghost.**

Сервисы вроде BotGhost хостят много приложений под одним IP, что приводит к частым Rate Limit. Не используйте BotGhost с нашим API.
{% endhint %}

<h3 align="center">Глобальный Rate Limit</h3>

| Категория эндпоинтов               | Лимит                                      |
| ---------------------------------- | ------------------------------------------ |
| Общие эндпоинты (`/stats/*`)       | **20 запросов/мин**                        |
| ER:LC эндпоинты (`/erlc/server/*`) | **2 запроса/сек** или **120 запросов/мин** |

<h4 align="center">Лимиты По Эндпоинтам</h4>

| Эндпоинт                   | Лимит               | Описание                                    |
| -------------------------- | ------------------- | ------------------------------------------- |
| `/erlc/server`             | 2/сек или 120/мин   | Информация о сервере                        |
| `/erlc/server/players`     | 2/сек или 120/мин   | Список игроков                              |
| `/erlc/server/joinlogs`    | 2/сек или 120/мин   | Логи входов/выходов                         |
| `/erlc/server/queue`       | 2/сек или 120/мин   | Очередь сервера                             |
| `/erlc/server/killlogs`    | 2/сек или 120/мин   | Логи убийств                                |
| `/erlc/server/commandlogs` | 2/сек или 120/мин   | Логи команд                                 |
| `/erlc/server/modcalls`    | 2/сек или 120/мин   | Вызовы модераторов                          |
| `/erlc/server/bans`        | 2/сек или 120/мин   | Список банов                                |
| `/erlc/server/vehicles`    | 2/сек или 120/мин   | Список транспорта                           |
| `/erlc/server/staff`       | 2/сек или 120/мин   | Список администрации                        |
| `/erlc/server/command`     | Очередь: 100 команд | Выполнение команд (не подвержен rate limit) |
| `/stats/*`                 | 20 запросов/мин     | Публичные эндпоинты статистики              |

> **Важно:** Лимит ER:LC эндпоинтов проверяется **одновременно** по двум параметрам:
>
> * Не более **2 запросов в секунду**
> * Не более **120 запросов в минуту**
>
> Превышение **любого** из лимитов приведёт к ошибке 429.

<h3 align="center">Защита От Cпама</h3>

Приложения, отправляющие **500+ запросов в секунду**, будут автоматически заблокированы.

<p align="center"><strong>Уровни Защиты От Спама</strong></p>

| Уровень     | Порог (запросов/сек) | Блокировка | Код ошибки                     |
| ----------- | -------------------- | ---------- | ------------------------------ |
| **Level 1** | 500+                 | 5 минут    | `spam_detected_level_1` (5020) |
| **Level 2** | 5000+                | 15 минут   | `spam_detected_level_2` (5021) |
| **Level 3** | 100000+              | 60 минут   | `spam_detected_level_3` (5022) |

<h4 align="center"><strong>Блокировка За Спам После Rate Limit</strong></h4>

Если после получения Rate Limit (429) клиент продолжает отправлять запросы:

| Поведение                  | Наказание           | Код ошибки                |
| -------------------------- | ------------------- | ------------------------- |
| >20 запросов/сек после 429 | Бан на **30 минут** | `rate_limit_abuse` (5016) |
| >50 запросов/сек после 429 | Бан на **1 час**    | `rate_limit_abuse` (5016) |

> **Совет:** При получении 429 немедленно прекратите запросы и дождитесь `retry_after`.

<h4 align="center"><strong>Ответ При Превышении Лимита (Rate Limit)</strong></h4>

```json
{
  "code": 5016,
  "message": "rate_limit_exceeded",
  "retry_after": 1,
  "details": {
    "second_count": 5,
    "minute_count": 125,
    "limits": {
      "per_second": 2,
      "per_minute": 120
    }
  }
}
```

<h4 align="center"><strong>Ответ При Блокировке За Спам После Rate Limit</strong></h4>

<h4 align="center"><strong>Заголовки Rate Limit</strong></h4>

| Заголовок             | Пример значения  | Описание                                          |
| --------------------- | ---------------- | ------------------------------------------------- |
| `RateLimit-Limit`     | 120              | Максимальное количество запросов до получения 429 |
| `RateLimit-Remaining` | 119              | Оставшееся количество запросов                    |
| `RateLimit-Reset`     | _Unix-timestamp_ | Unix-время, когда лимит будет сброшен             |

<h4 align="center">Очередь Команд</h4>

Эндпоинт `/erlc/server/command` использует систему очереди:

* **Максимальный размер очереди:** 100 команд
* **Интервал выполнения:** 1 команда каждые 1.5 секунды

Когда очередь заполнена, вы получите:

```json
{
  "code": 5015,
  "message": "CommandQueueFull"
}
```
