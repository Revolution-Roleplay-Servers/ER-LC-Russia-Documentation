<h3 align="center">Новые Регламенты и Правила</h3>

> **Важное примечание:** Сейчас у лидеров начинается **первый ознакомительный срок (без дня блата)**. Правила будут дорабатываться в процессе, а с введением системы Правительства (СК, Прокуратуры и Суда) контроль за соблюдением регламентов станет ещё строже и реалистичнее.

#### 2. **Расширенная Система Наказаний**

* **Таймеры Наказаний** — автоматическое снятие по истечении срока.
* **Роли Званий** — теперь для каждого звания при повышении вы сможете настроить роли, которые автоматически будут выдаваться участнику при повышении на это звание.
* **Подразделения** — теперь вы можете настроить подразделения для вашего департамента. Это позволит удобно повышать сотрудников. В подразделениях также можно определить роли, которые будут назначаться при их принятии или переводе.

#### 3. **Система Отстранений**

* **Новый тип наказания** для удобного отстранения сотрудников.
* **Срок отстранения** с автоматическим снятием.
* **Объяснительные** — теперь участник сможет написать объяснительную через бота, чтобы оправдать свои нарушения или сознаться в них.
* **Прикрепление доказательств к объяснительной** через удобное поле в модальном окне.
* **Уведомления в ЛС** — отправка объяснительной через ЛС.

#### 4. **Синхронизация Ролей**

* **Последовательный цикл** каждые 40 минут:
  1. Проверка менеджмента.
  2. Проверка прав доступа к каналам запроса ливери.
  3. Синхронизация ролей фракций с сервером РП сцен.
* **Автоматическая выдача/удаление** ролей
* **Поддержка 8+ фракций:** МВД, ФСБ, МЧС, МЗ, Суд, СК, Департамент Транспорта

#### 5. **Система Конфигурации**

* **Настройка Прав** — разрешённые роли для команд бота.
* **Настройка Каналов** — каналы для наказаний, повышений, принятий, увольнений и т.д.
* **Типы Увольнений** — категории с ролями для автоматической выдачи.
* **Роли Наказаний** — роли, которые будут выдаваться по системе наказаний.
* **Запрещённые Роли** — роли, на которые нельзя будет повысить (например роль "Участник")
* **Режим Подразделений** — если куратор вашей фракции одобрил возможность сотрудников работать в нескольких подразделениях, вы можете активировать эту функцию. Это позволит боту разрешить прием сотрудников в несколько подразделений.
* **Управление Префиксами** — мы создали удобную панель для управления префиксами сотрудников вашей фракции. Теперь вы можете легко добавлять префиксы, удалять их из ников и многое другое (подробнее пункт 6).

#### 6. **Система Префиксов**

* **Уникальные префиксы** вида `[XX-001]`.
* **Автоматическая проверка** на уже существующий префикс.
* **Несколько префиксов** у одного пользователя.
* **Основной префикс** — отображаемый в нике.
* **Удаление префиксов** через панель управления префиксами.

#### 7. **Прочее**

* Оптимизировали бота.
* Сделали его применение более автоматизированым.
* Учли ваши идеи.
* Улучшили систему модколлов (теперь модераторы смогут принять модколл и вам придёт сообщение в ЛС).
