# 👋 Вступление

<h2 align="center"><strong>Moscow RolePlay — Правила Персонала</strong></h2>

<p align="center">Правила персонала — это фундамент дисциплины и порядка, определяющий обязанности персонала по созданию честной игровой среды. Он запрещает использование полномочий в личных целях и требует от команды вежливого, объективного отношения к игрокам. Соблюдение этих норм гарантирует стабильную работу сервера и высокий уровень доверия со стороны сообщества.</p>
